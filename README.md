# HIIT Timer

タバタ式インターバルトレーニング用タイマー。ブラウザで動作するシングルHTMLファイル。

## 機能

- **インターバル設定** — 運動時間・休憩時間（5秒刻み）・セット数（最大20）を自由に変更
- **デフォルト設定** — タバタ式（20秒 × 10秒 × 8セット）
- **3・2・1 カウントダウン** — スタート前の準備カウント
- **二重リング（requestAnimationFrame 60fps）**
  - 外側リング: プログラム全体の進捗を WORK（オレンジ）/ REST（シアン）で按分済みのセグメントとして常時表示
  - 内側リング: 現フェーズの残り時間をリアルタイム表示（フェーズ切替ごとに 12時→12時 でリセット）
- **フェーズ表示** — WORK / REST を色で即座に判別（オレンジ / シアン）
- **残り時間表示** — プログラム全体の残り秒数を常時表示
- **セットドット** — 現在位置と完了済みセットを視覚的に確認
- **コントロール** — 一時停止・再開・スキップ・リセット
- **ビープ音** — 残り3秒カウント・フェーズ切替・終了時に効果音
- **リザルト画面** — 終了後に総時間・セット数を表示
- **ワークアウト履歴** — 最大30件をLocalStorageに自動保存
- **設定パネル（歯車アイコン）** — アクセントカラー（WORK / REST）・モチベーションテキスト・サウンドのオン/オフ
- **ダーク / ライト / オートモード** — 右上のボタンで切替、設定はブラウザに記憶

## 技術スタック

- HTML / CSS / JavaScript（ビルドツール不要）
- React 18（CDN経由、unpkg）
- Babel Standalone（JSX変換）
- Web Audio API（ビープ音）
- LocalStorage（設定・履歴の永続化）
- `requestAnimationFrame` + `performance.now()` による 60fps スムーズアニメーション

## リング仕様

```
例: WORK 5s, REST 5s, 2セット (計 15s)

インナー(位相別リセット):
  WORK: 12時 → 12時 (0→1 phaseElapsedF)
  REST: 12時 → 12時 (0→1 phaseElapsedF)
  WORK: 12時 → 12時 (0→1 phaseElapsedF)

アウター(プログラム全体を按分・セグメント固定):
  WORK: 12時 → 4時
  REST:  4時 → 8時
  WORK:  8時 → 12時 (消費済みセグメントは順に消えていく)
```

## ローカルで開く

```bash
open index.html
```

※ ローカルファイルでの音声再生に制限があるブラウザでは、ローカルサーバー経由で開いてください。

```bash
npx serve .
# または
python3 -m http.server 8080
```
